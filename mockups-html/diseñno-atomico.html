Tipografía

Usamos la tipografía Poppins como fuente principal para mantener una apariencia profesional y legible.

Título H1 (Poppins)

Usado para títulos principales de página: h1

Título H2 (Poppins)

Usado para secciones importantes: h2

Título H3 (Poppins)

Usado para subsecciones: h3

Título H4

Usado para títulos de componentes: h4

Título H5

Usado para títulos en tarjetas: h5

Título H6

Usado para subtítulos en componentes: h6

Texto de párrafo regular. Diseñado para tener buena legibilidad en textos largos. Este texto está en negrita y este está en cursiva.

Texto de párrafo: p

Texto pequeño usado para metadatos, fechas, anotaciones y contenido secundario.

Texto pequeño: small

Texto destacado usado para introducciones o contenido importante.

Texto destacado: p.lead

Texto atenuado para información secundaria o contextual.

Texto atenuado: .muted

Páginas

Implementaciones concretas de plantillas con contenido real.

Las páginas principales de la aplicación son:

  • Login: Página de inicio de sesión para usuarios.
  • Dashboard: Vista general con KPIs y acciones rápidas.
  • Unidades: Gestión de unidades/departamentos.
  • Emisiones: Creación y gestión de cargos mensuales.
  • Pagos: Registro y conciliación de pagos.
  • Personas: Gestión de propietarios y residentes.
  • Configuración: Ajustes del sistema.
